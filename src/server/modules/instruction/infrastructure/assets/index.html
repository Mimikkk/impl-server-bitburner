API - Bitburner

This is the API for the Bitburner server. If you are able to see this page, the server is running.

Connecting to the server

  1. Enter the hostname of the server in the "Hostname" field.
  2. Enter the port of the server in the "Port" field.
  3. Click "Connect".

You should now see "connected" as the status.
